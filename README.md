# Towers of Hanoi

## Overview
The Towers of Hanoi is a classic puzzle game that challenges players to move a stack of disks from one peg to another, following specific rules. The objective is to move all disks to the rightmost peg in the minimum number of moves.

## How to Play
1. **Game Setup**: The game starts with a number of disks stacked in ascending order of size on the leftmost peg.
2. **Moving Disks**: 
   - Click on a disk to pick it up.
   - Click on a peg to place the disk down.
   - You can only move one disk at a time.
   - A larger disk cannot be placed on top of a smaller disk.
3. **Winning the Game**: Move all disks to the rightmost peg to win. Try to solve the puzzle in the minimum number of moves!

## Features
- Interactive gameplay with visual feedback.
- Timer and move counter to track your progress.
- Achievements for completing the game under certain conditions.
- Difficulty settings to adjust the number of disks.

## Installation
To run the Towers of Hanoi game, simply open the `src/towersofhanoi.html` file in a web browser. No additional setup is required.
